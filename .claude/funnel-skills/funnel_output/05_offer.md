## 3. La garantía

> **Nota:** No aplica garantía de devolución tipo e-commerce, pero sí se puede ofrecer garantía de tranquilidad que elimine el riesgo percibido de Carolina.

### Garantía "Sin Riesgo de Clima"

```
🌧️ GARANTÍA TRANQUILIDAD:

Si el día de tu evento el clima no permite disfrutar al aire libre,
reprogramamos tu celebración a otra fecha sin ningún costo adicional.
Tu anticipo queda 100% protegido.

No pierdes tu plata. No pierdes tu plan. Solo cambias la fecha.
```

### Garantía "Visita Primero"

```
👀 ¿NO CONOCES EL LUGAR?

Ven a conocer Laguna Escondida antes de reservar.
Almuerza con tu familia un domingo, recorre el espacio
y decide si es el lugar para tu evento.

Si después de conocerlo decides reservar, te descontamos
el valor de tu almuerzo del precio del evento.

Cero riesgo. Primero conoces, después decides.
```

**Recomendación:** Usar ambas. La garantía de clima elimina la objeción #1 de Carolina. La visita previa convierte prospectos indecisos en clientes — una vez ven el lago y prueban la comida, la reserva se cierra sola.

## 4. Urgencia / Escasez (REAL)

### Escasez real:
Solo hay **1 kiosco privado** y **1–2 zonas semi-privadas**. Máximo 2 eventos por fin de semana. Esto es escasez genuina.

### Copy para WhatsApp (cuando hay interés):
```
📅 Te cuento: los fines de semana solo podemos hacer 1-2 eventos
porque reservamos el espacio completo para cada grupo.

[Mes actual] ya tiene [X] fechas ocupadas.
Si ya tienes fecha en mente, mejor apartarla con el anticipo
para que no se cruce con otra reserva.

¿Qué fecha estás pensando?
```

### Copy para stories (recurrente, cada semana):
```
📅 Disponibilidad de eventos — [Mes]:

✅ Sábado [fecha]: Disponible
❌ Domingo [fecha]: Reservado
✅ Sábado [fecha]: Disponible
✅ Domingo [fecha]: Disponible
❌ Sábado [fecha]: Reservado

Los fines de semana se van llenando.
Escríbenos para apartar tu fecha 👉 [link WhatsApp]
```

**Nota:** Publicar disponibilidad real cada semana cumple doble función — genera urgencia honesta Y demuestra que otros ya reservan (prueba social implícita).

## 6. Paquetes sugeridos (para simplificar la cotización)

> **Nota estratégica:** Tener paquetes predefinidos reduce fricción. Carolina no quiere cotización personalizada que tarde 3 días — quiere saber cuánto cuesta YA.

### Paquete Básico — "Celebración al Lago"
- **Grupo:** 10–20 personas
- **Incluye:** Zona semi-privada, menú básico (plato fuerte + bebida), pesca deportiva, parqueadero
- **Precio:** $300,000 COP
- **Ideal para:** Reuniones familiares, cumpleaños íntimos

### Paquete Completo — "Fiesta Campestre"
- **Grupo:** 20–35 personas
- **Incluye:** Kiosco privado, menú completo (entrada + plato fuerte + postre + bebidas), pesca deportiva, configuración de mesas, atención dedicada
- **Precio:** $500,000–$600,000 COP
- **Ideal para:** Cumpleaños infantiles, revelaciones de género, baby showers

### Paquete Premium — "Laguna Exclusiva"
- **Grupo:** 35–50 personas
- **Incluye:** Todo el paquete completo + exclusividad de zona (no se reciben otros clientes en horario del evento), menú premium, decoración base incluida
- **Precio:** $700,000–$800,000 COP
- **Ideal para:** Eventos grandes, celebraciones de empresa, eventos especiales

### Copy para presentar paquetes en WhatsApp:
```
🎉 Tenemos 3 opciones para tu evento:

1️⃣ CELEBRACIÓN AL LAGO ($300K) — Hasta 20 personas
   Zona semi-privada + menú + pesca deportiva

2️⃣ FIESTA CAMPESTRE ($500K–$600K) — Hasta 35 personas
   Kiosco privado + menú completo + atención dedicada

3️⃣ LAGUNA EXCLUSIVA ($700K–$800K) — Hasta 50 personas
   Todo lo anterior + exclusividad de zona + menú premium

Todos incluyen parqueadero, pesca deportiva con 5 especies
de peces y la experiencia "pesca tu comida" 🎣

¿Cuántas personas serían para tu evento? Así te recomiendo
el paquete ideal 😊
```
